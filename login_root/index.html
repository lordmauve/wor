World of Rodney Account Management

We use OpenID for login. This means that if you have an account with any of these websites, you can just start playing:
  • World of Rodney
  • Google
  • GMail/Googlemail
  • Facebook
  • Flickr
  • Wordpress

If you don't have an account with any of the above, or want to create a new local account, you can sign up here: Create a new World of Rodney account
